KJumpingCubes

Description: This is a two-person pure strategy game, involving no element of chance. The winning state is once the entire N x N board is filled with your color. This game can have either two human players, human vs. computer, or two computer players. The computer is based on a depth four Minimax strategy, following a simple heuristic model. Traveling Salesman Problem (TSP)

Description: A final project for my Algorithms course, which was a competition to solve instances of an NP-complete problem. In a team of three, we were attempting to find the best approximation for a TSP path problem where each city has a color of Red or Blue, and the path returned cannot have more than three of the same color consecutively. We modified the Christofides approximation algorithm (which guarantees getting within 1.5x the optimal solution) but forgot that we couldn't guarantee the triangle inequality. It was a failed attempt, but a good learning experience overall.
